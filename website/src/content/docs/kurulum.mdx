`fatura` paketini Node.js / TypeScript projenize eklemek için:

```bash
npm install fatura
```

Ardından istemciyi oluşturup kullanmaya başlayabilirsiniz:

```ts
import { createFaturaClient } from "fatura"

const client = createFaturaClient("TEST")
```

> TEST ortamında doğrulama yapmadan PROD ortamında malî işlem başlatmayın.
