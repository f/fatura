## getToken

Portal oturumu için token alır.

```ts
getToken(userName: string, password: string): Promise<string>
```

```ts
const token = await client.getToken("KULLANICI", "PAROLA");
```

## logout

Aktif token ile oturumu sonlandırır.

```ts
logout(token: string): Promise<unknown>
```
