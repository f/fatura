## getInvoiceHTML

```ts
getInvoiceHTML(token: string, uuid: string, options: { signed: boolean }): Promise<string>
```

İmzalı veya taslak faturanın HTML çıktısını döndürür.

## getDownloadURL

```ts
getDownloadURL(token: string, invoiceUUID: string, options: { signed: boolean }): string
```

GİB indirme uç noktasına yönlendiren bağlantıyı metin olarak döndürür.
