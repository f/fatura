## getUserData

```ts
getUserData(token: string): Promise<UserData>
```

Portal profil verisini `UserData` tipine dönüştürerek döndürür.

## updateUserData

```ts
updateUserData(token: string, userData: UserData): Promise<unknown>
```

Profil alanlarını GİB tarafına kaydeder.
